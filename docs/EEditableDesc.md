## API

#### EEditableDesc 属性

| 属性名   | 说明             | 类型    | 默认值                                  | 版本  |
| -------- | ---------------- | ------- | --------------------------------------- | ----- |
| value    | 双向绑定         | Array   | '{label: '', value: '', editable: false}' | 0.0.4 |
| col      | 自定义 x 列      | Number  | 2                                       | 0.0.4 |
| editable | 是否开启编辑模式 | Boolean | false                                   | 0.0.4 |
