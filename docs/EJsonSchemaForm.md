## API

#### EJsonSchemaForm 属性

| 属性名        | 说明                          | 类型    | 默认值 | 版本  |
| ------------- | ----------------------------- | ------- | ------ | ----- |
| value         | 原始值                        | String  | -      | 2.4.0 |
| errorMessages | 错误信息，需要与 ajv 配套使用 | object  | {}     | 2.4.0 |
| isReadOnly    | 是否只读                      | Boolean | false  | 2.4.0 |
| schema        | json schema 描述              | String  | -      | 2.4.0 |
