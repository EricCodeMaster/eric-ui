## API

#### ETagInput 属性

| 属性名         | 说明                              | 类型    | 默认值     | 版本  |
| -------------- | --------------------------------- | ------- | ---------- | ----- |
| value          | 值                                | Array   | []         | 2.4.0 |
| max            | 最多可用标签，为 0 时不限制       | Number  | 0          | 2.4.0 |
| readonly       | 只读                              | Boolean | false      | 2.4.0 |
| inputMaxlength | 单标签最大输入字符，为 0 时不限制 | Number  | 0          | 2.4.0 |
| text           | 新增按钮的展示文本                | String  | '增加新项' | 2.4.0 |
